Culinary Web App

Live Demo
Source Code

Web app that allow users to explore different countries' cuisines. The front-end was designed using Java-Script based framework, Next.js with React components. Firebase Authentication was used as backend service to allow users to log in or sign up. Firebase's Cloud Firestore was used as NoSQL backend database to store user's data and cuisines' data. CICD practice was used to build and deploy docker container to AWS EC2 server. This project is work in progress and still adding new features like, save recipes and creating recipes.

Smartphone Controlled LED Lights

The purpose of this project was to develop an interior lighting system for the car with LED strip light. Developed the circuit, using LTspice software and build the circuit, using Electronic components. The circuit's ouput was controlled using Arduino microcontroller, which was programmed in C++ itself. Bluetooth module was used to communicate with circuit to control lighting system.

Desktop application for Excel Files

Source Code

Developed a desktop application to read Excel Files from desktop and display on screen. The purpose of this application is view excel file data without the need of Microsoft Excel. Leveraged python and it's library, tkinter tollkit, to program Graphical User Interface (GUI) and used openpyxl library to read Excel data.
